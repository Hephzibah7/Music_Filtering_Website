Query Guidelines

Please enter your query in the following format:

→ The query should be written in attribute-value pair(attribute = value)

→ Each attribute should be in double quotes

→ The corresponding value should also be in double quotes

→ All the attributes should be written in block letters

→ Use spaces between each object(word, operator, etc)

→ There should be at least one conjunction(and, or, not) between two attributes
